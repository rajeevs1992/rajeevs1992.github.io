The Department of CSE of Government Engineering College has undergone a makeshift change towards improving the quality of labs by incorporating an automated judge for program validation.

The labs are now invigilated by the popular Programming Contest Engine, named Mooshak. As a starter, the S4 data structure labs are now conducted using the Mooshak, largely due to the enthusiasm by Mr Vipin, Asst. professor, GEC Thrissur, who asked to start using the software right away. The DS lab exams are to be conducted on the same platform.

The result was overwhelming.Many of the students managed to come up with high quality programs for algorithms like the Dijkstra's, which were tested with inputs as large as 1.2MB in size, a 1000 node graph.The programs get rejected if they run too long.

This program will further be expanded to S1S2 C labs, S3 C labs and S5 Programming Paradigms labs.
