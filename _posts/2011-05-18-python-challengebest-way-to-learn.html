For the last few weeks I am busy with the best coding experience that I ever had in my life.One of my friends said about this wonderful site which is an online puzzle with a difference.......

For those who are new to the Python challenges,it is just a programming puzzle which requires no registration,no tiring mail sending and absolutely no unclear rejection emails as from Facebook puzzles.If you happen to solve a puzzle,you will get a URL of the next puzzle with no hesitation or code efficiency issues/checks

And there is no necessity that the programs must be done with Python,but using Python will prove to be easiest.But there are some puzzles which use some formats/data which solely belong to python alone.
